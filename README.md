# CODESOFT - Number Guessing Game

## Description
A simple **Number Guessing Game** written in C++ that demonstrates basic input/output and control flow.  
The program generates a random number between **1 and 100**, and the player tries to guess it. After each guess, the program tells the player if their guess is too low, too high, or correct.

This project is perfect for beginners to understand:
- Basic C++ syntax
- Loops and conditional statements
- Handling user input and validation
- Random number generation
